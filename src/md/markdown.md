# 마크다운 문법

마크다운은 문서를 작성할 때 정말 실용적인거 같다. 상대적으로 html 태그 다는게 복잡하게 느껴진다. 뭔가 사람에게도 쉽게 읽히고 컴퓨터에도 읽힐수있는 언어 같다. 제목을 쓸때 <h1></h1> 이런 태그가 아닌 #, ##, ### 이렇게 표현을 하고 중요한 내용은 `**중요한 내용 **` 으로 표시한다.
문서의 의도를 빠르게 파악할수 있을거같다

마크다운은 개발문서에도 꼭 필요한 언어라고 느껴지고 과제를 할때도 훨씬편하게 할수있을거같다.

<li></li> 같은 태그를 사용하지 않고 - 이나 숫자로 간단하게 표현할 수 있고, 링크또한 [구글](https://google.com) 형태로 직관적으로 작성할 수 있다. 
이분야에 전문성을 가지고 있지않아도  어느정도 이해가 가능할것같다.
